🔐 Authentication
Pass your API key in every protected request header:
X-API-Key: your-secret-api-key # OR as query param: GET /api/stats?api_key=your-secret-api-key
POST Create Order
POST /api/create_order
Create a new payment order. Returns unique penny amount + QR + pay_url.
# Request curl -X POST https://yourdomain.com/api/create_order \ -H "X-API-Key: your-key" \ -H "Content-Type: application/json" \ -d '{"amount": 100, "label": "Order #42"}' # Response { "order_id": "A1B2C3D4", "amount": 100, "unique_amount": 100.03, "status": "PENDING", "pay_url": "https://yourdomain.com/pay/A1B2C3D4", "qr_base64": "iVBORw0KGgo...", "created_at": "2026-04-16 14:42:01" }
GET Order Status (Public)
GET /api/order/:order_id
No API key needed. Use this for customer-side polling.
curl https://yourdomain.com/api/order/A1B2C3D4 # Response when SUCCESS: { "status": "SUCCESS", "paid_by": "Mr PANKAJ PANDIT", "paid_at": "2026-04-16 14:45:22", "unique_amount": 100.03 }
GET Stats
GET /api/stats
curl -H "X-API-Key: your-key" https://yourdomain.com/api/stats { "total": 42, "success": 38, "pending": 3, "cancelled": 1, "total_collected": 4820.50 }
POST Cancel Order
POST /api/cancel/:order_id
curl -X POST -H "X-API-Key: your-key" \ https://yourdomain.com/api/cancel/A1B2C3D4
POST Update Config
POST /api/config
curl -X POST -H "X-API-Key: your-key" \ -H "Content-Type: application/json" \ -d '{"upi_id": "newid@upi", "upi_name": "New Name"}' \ https://yourdomain.com/api/config
☁️ Cloudflare Setup
To expose your Termux server via Cloudflare Tunnel:
# Install cloudflared in Termux pkg install cloudflared # Start tunnel (no account needed - quick tunnel) cloudflared tunnel --url http://localhost:5000 # OR with custom domain (needs CF account): cloudflared tunnel login cloudflared tunnel create fampay-gw cloudflared tunnel route dns fampay-gw pay.yourdomain.com cloudflared tunnel run fampay-gw
